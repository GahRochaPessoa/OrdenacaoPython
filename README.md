Pequeno trabalho de Algoritmos e Estruturas de dados envolvendo os métodos de ordenação QuickSort e BubbleSort, ambos implementados em Python.
Os testes estão sendo gerados com um vetor de 10000 posições com números aleatórios de 1 à 100000. 
[Aviso!] Pode ocorrer algum problema de overflow, ao tentar rodar o bubbleSort e depois o quickSort ou vice-versa.
